# Product

Keyorix is a secrets management platform focused on:
- discovering secrets across systems and code
- securely storing and managing secrets
- enabling AI-driven workflows via MCP

---

## Core Capabilities

1. Secret Discovery
- scan codebases, files, and environments
- detect hardcoded or leaked secrets

2. Secret Storage
- encrypted storage (see encryption.md)
- centralized management

3. MCP Integration
- natural language interface to interact with secrets
- commands like:
  - "show all secrets"
  - "scan project"

4. SDK Ecosystem
- Node, Python, Go, Java
- integration into developer workflows

---

## Target Users

- developers
- DevOps / SRE teams

---

## Current Focus

- MCP + AI workflows
- secret scanning
- local-first usability

## Constraints

- minimize complexity
- backend is source of truth
- security over convenience
